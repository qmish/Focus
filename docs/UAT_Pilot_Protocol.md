# UAT Protocol for Pilot Group

Дата: 25 марта 2026 г.

## Цель

Провести формализованную пользовательскую приемку перед go-live.

## Состав пилотной группы

- 5-10 бизнес-пользователей (основные сценарии чата/звонков)
- 2 администратора (управление пользователями/конференциями/наблюдаемость)
- 1 представитель security/compliance (проверка критичных политик)

## Сценарии UAT (минимум)

1. Auth + вход в систему.
2. Создание комнаты и обмен сообщениями.
3. Присоединение к звонку (Jitsi embed flow).
4. Bot-команды (`/status`, `/create`, `/schedule`) в рабочем контуре.
5. Admin сценарии (stats/users/conferences/observability).
6. Проверка ошибок и восстановление после reconnect.

## Критерии успешности

- Не менее 95% сценариев успешно.
- Нет blocker-критичности (`SEV-1/SEV-2`) по итогам UAT.
- Все найденные дефекты классифицированы и занесены в backlog.
