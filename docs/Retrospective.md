## ⚠️ Проблемы и решения

### Проблема 1: Microsoft Graph API SDK
**Проблема:** Новая версия SDK имеет breaking changes  
**Решение:** Использованы заглушки для методов Update/Delete с пометкой TODO

### Проблема 2: WebSocket тестирование
**Проблема:** Сложность тестирования WebSocket в k6  
**Решение:** Создан отдельный стресс-тест для WebSocket

### Проблема 3: E2E тесты без реального Keycloak
**Проблема:** E2E тесты требуют работающий Keycloak  
**Решение:** Mock-авторизация для CI, полный тест для staging

## 🎓 Извлечённые уроки

### Технические
1. **Go + GORM** — отличное сочетание для API
2. **React + Zustand** — просто и эффективно
3. **Kubernetes HPA** — требует тщательной настройки метрик
4. **OWASP ZAP** — находит реальные уязвимости

### Процессные
1. **Документация важна** — экономит время в будущем
2. **Тесты окупаются** — меньше багов в production
3. **Частые релизы** — быстрее обратная связь
4. **Автоматизация CI/CD** —Must-have для проекта

### Управленческие
1. **Реалистичные оценки** — лучше заложить +20% времени
2. **Приоритизация** — сначала критический функционал
3. **Коммуникация** — регулярные обновления статуса
