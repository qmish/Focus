# Bots

**Версия:** 1.0  
**Дата:** 27 марта 2026 г.

---

## 1. Назначение

`BotEngine` обрабатывает slash-команды в комнатах и публикует ответы как обычные сообщения от системного пользователя бота.

---

## 2. Компоненты

- `internal/bots/bots.go`
  - парсинг команд
  - rate limiting
  - вызов handler команд
  - сохранение и доставка ответа
- `bot_command_events`
  - аудит и наблюдаемость выполнения команд
- зависимости:
  - `RoomRepository`
  - `UserRepository`
  - `MessageRepository`
  - `WebSocket hub`
  - `CalendarService` (для `/schedule`)

---

## 3. Встроенные команды

- `/help` — список команд
- `/status` — краткий статус движка
- `/create` — создать комнату/встречу
- `/schedule` — запланировать встречу
- `/members` — участники комнаты
- `/whoami` — информация о пользователе
- `/dice` — случайный бросок
- `/find` — поиск комнат

---

## 4. Поток выполнения

1. Сообщение начинается с `/`.
2. Проверка доступа пользователя к комнате.
3. Проверка rate-limit для пользователя.
4. Вызов handler команды.
5. Сохранение bot message в БД.
6. Broadcast сообщения в websocket.
7. Запись события выполнения в `bot_command_events`.

---

## 5. Ошибки

- Ошибки команд возвращаются как человекочитаемые bot-ответы.
- Ошибки доставки/сохранения не ломают основной message flow.
- Все критичные сбои фиксируются в command events для admin-наблюдаемости.
